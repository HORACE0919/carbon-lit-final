# 碳治理文献日报

> 农林经济管理博士研究助手 · 四级漏斗精准筛选系统

每天北京时间 **08:00** 全自动运行，零人工干预。

## 四级漏斗流程

| 级别 | 内容 | 规模 | Token消耗 |
|------|------|------|-----------|
| 第1级 | CrossRef免费海选（公开元数据） | ~80篇 | 零成本 |
| 第2级 | 多维打分初筛（相关性/分区/可实现性/引用/时效） | 12篇 | 零成本 |
| 第3级 | AI分级精读（OA文献深度解析） | OA部分 | 付费token边界 |
| 第4级 | 新锐分区顶刊终选（分区优先，OA不参与排序） | 3篇 | 零成本 |

## 核心原则

- **OA中立**：OA状态不参与打分排序，高分区非OA永远优先于低分区OA
- **分区保底**：唯一使用新锐分区表作为期刊层级标准
- **Token边界**：付费token严格限定在初筛12篇内
- **容错优先**：单条异常直接跳过，绝不中断整体流程

## 一次性配置

1. Fork 本仓库
2. Settings → Secrets → Actions → New secret
   - `SILICONFLOW_API_KEY` = 你的硅基流动 Key（cloud.siliconflow.cn 免费注册）
3. Settings → Pages → Source 选 `main` 分支
4. Actions → 每日文献更新 → Run workflow（首次手动触发）
