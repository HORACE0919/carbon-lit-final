# 碳治理文献日报

> 农林经济管理博士研究助手 · 四级漏斗精准筛选系统

每天北京时间 **07:00** 全自动运行，也可从页面的「立即更新日报」进入 Actions 手动触发。

## 四级漏斗流程

| 级别 | 内容 | 规模 | Token消耗 |
|------|------|------|-----------|
| 第1级 | 中文+国际独立检索（Crossref + OpenAlex） | 每组4个查询×双源 | 零成本 |
| 第2级 | 多维综合得分初筛 | 12篇 | 零成本 |
| 第3级 | 已核验公开PDF全文解读；其余只解读摘要 | 初筛12篇 | 付费token边界 |
| 第4级 | 综合得分终选（分区已纳入综合得分，语言/OA不额外加减分） | 4篇 | 零成本 |

## 核心原则

- **OA中立**：OA状态不参与打分排序；分区、相关性、方法、引用与时效统一进入综合得分
- **双语海选**：中文与国际文献独立检索，但不设语言配额；初筛与终选统一按综合得分排序
- **分区保底**：使用用户提供的2026新锐分区表（22,281条）；冲突刊名明确标注待核验
- **阅读边界**：只有已核验公开PDF才自动取全文；机构权限文献在云端只能基于公开摘要
- **Token边界**：付费token严格限定在初筛12篇内，不因OA状态改变推荐排名
- **真实降级**：单源失败可由另一源补位；所有源失败则明确报错，不生成伪成功日报

## 一次性配置

1. Fork 本仓库
2. Settings → Secrets → Actions → New secret
   - 优先：`GEMINI_API_KEY` = 你已配置的 Gemini Key
   - 备用：`SILICONFLOW_API_KEY` = 硅基流动 Key
   - 两者都存在时优先使用 Gemini；都不存在时使用明确标注的规则降级，不会留空
3. Settings → Pages → Source 选 `main` 分支
4. Actions → 每日文献更新 → Run workflow（首次手动触发）
